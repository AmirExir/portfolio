🛠️ PSS/E IDV Generator – Transmission Line Upgrade Tool

This GUI automates the generation of PSS/E `.idv` files for proposed transmission line upgrades. Given bus numbers, voltage level, conductor type, and length, it calculates impedance, MVA ratings, and produces ready-to-import `.idv` entries using custom CSV lookup tables.

🔧 Key Features

  • Reads impedance and rating data from CSV files
  • Supports multiple conductor types and voltage levels
  • Calculates MVA rating based on user-defined ampacity
  • Outputs PSS/E `.idv` lines with correct syntax and formatting
